# Workshop

A collection of creative projects and experiments in web development. Explore interactive visualizations and algorithmic art.

## Projects

- **Polar Clock**: An interactive clock that displays time using polar coordinates, creating a mesmerizing radial pattern that evolves with each second, minute, and hour.
- **Conway's Game of Life**: Explore the classic cellular automaton where simple rules give rise to complex, emergent behaviors. Watch patterns evolve, die out, or stabilize in this simulation of artificial life.
- **House Planner**: A drag-and-drop tool for designing room layouts. Add furniture, walls, and other elements to visualize and plan your ideal living space on an interactive grid.
- **Barfoo**: A music library browser that scans your local music directory and displays albums organized by artist, providing a simple interface to explore your collection.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

This project is built with [Next.js](https://nextjs.org) and showcases various web development techniques.
